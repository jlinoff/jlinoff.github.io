Module: accordion

Support for accordions. Accordions are composed of two parts: an accordion toggle button and an accordion panel. When the button is clicked it toggles the display the panel which contains the contents. Accordions are the primary idiom used to display information on the app pages.

(static) makeAccordionEntry(title, panel, expandAction, collapseAction)

Make an accordion entry.

An accordion entry is composed of a button and a panel. The button and panel elements have class designations to make them easier to find.

The button has an icon the far left that changes to show whether the accordion entry is expanded or collapsed.

Parameters:
Name Type Description
title string

The accordion entry title displayed in the button.
panel element

The panel DOM element.
expandAction function

The action to take when the entry is expanded.
collapseAction function

The action to take when the entry is collapsed.

let accordionEntry = makeAccordionEntry(

                    .xInnerHTML('Choose the algorithm to use for encryption and decryption.'),
