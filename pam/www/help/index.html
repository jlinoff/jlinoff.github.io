Overview

PAM is single page web app that runs in your browser in a secure context that enables you to securely manage your personal account data as searchable records in a secure, flexible, and persistent manner without ever having to visit a server because it stores the data in a file that you control. The file is encrypted so even if it were stolen the data would be safe if the password you used to encrypt it was strong. It is available here: https://jlinoff.github.io/pam.
